# 符号说明与模型假设

## 一、符号说明

### 通用统计与评估指标

| 符号 | 含义 | 说明 |
|------|------|------|
| AUC | ROC曲线下面积 | 衡量模型区分正负样本的能力，0.5=随机，1.0=完美 |
| PR-AUC | 精确率-召回率曲线下面积 | 衡量不平衡数据下模型的排序能力 |
| ROC | 受试者工作特征曲线 | 以假阳性率为横轴、真阳性率为纵轴的曲线 |
| TP | 真阳性 (True Positive) | 实际患病且被正确预测为患病的样本数 |
| TN | 真阴性 (True Negative) | 实际未患病且被正确预测为未患病的样本数 |
| FP | 假阳性 (False Positive) | 实际未患病但被错误预测为患病的样本数 |
| FN | 假阴性 (False Negative) | 实际患病但被错误预测为未患病的样本数 |
| PPV | 阳性预测值 (Positive Predictive Value) | TP / (TP + FP)，预测为阳性中真正患病的比例 |
| NPV | 阴性预测值 (Negative Predictive Value) | TN / (TN + FN)，预测为阴性中真正未患病的比例 |
| F1 | F1分数 | 精确率与召回率的调和平均数，F1 = 2×PPV×Sens / (PPV+Sens) |
| F2 | F2分数 | 加权F分数（β=2），召回率权重为精确率的2倍 |
| Sens | 敏感度（灵敏度） | TP / (TP + FN)，实际患病中被正确检出的比例 |
| Spec | 特异度 | TN / (TN + FP)，实际未患病中被正确识别的比例 |
| Youden | Youden指数 | Sens + Spec − 1，用于选择最优阈值 |
| RMSE | 均方根误差 | 预测值与真实值偏差的平方与样本数比值的平方根 |
| R² | 决定系数 | 衡量回归模型对数据的拟合优度 |
| adj_R² | 调整决定系数 | 经自变量个数调整后的R² |
| AIC | 赤池信息准则 | 模型选择指标，权衡拟合优度与模型复杂度 |
| BIC | 贝叶斯信息准则 | 类似AIC但惩罚更严 |
| GCV | 广义交叉验证得分 | GAM模型的拟合优度指标 |
| UBRE | 无偏风险估计量 | GAM模型选择准则，类似AIC |
| EDF | 有效自由度 (Effective Degrees of Freedom) | GAM中各平滑项消耗的自由度 |
| VIF | 方差膨胀因子 | 衡量多重共线性的指标，VIF>10视为严重共线性 |
| CI | 置信区间 (Confidence Interval) | 通常使用95%置信区间 |
| SD | 标准差 (Standard Deviation) | 衡量数据的离散程度 |
| OR | 胜算比 (Odds Ratio) | Logistic回归中的效应量 |
| Log(OR) | 对数胜算比 | GAM偏效应图的纵轴单位 |
| RMSE | 均方根误差 | 回归模型的预测误差 |
| DCA | 决策曲线分析 | 评估模型在不同阈值下的临床净收益 |
| CV | 交叉验证 (Cross Validation) | 将数据分折训练与验证的评估方法 |
| OOF | 折外预测 (Out-of-Fold) | 交叉验证中每个样本由未参与训练的模型预测 |

## 二、模型假设

### 1. GAM（广义可加模型）基本假设

**1.1 可加性假设**
各预测变量对响应变量的效应在链接函数尺度上是可加的：
```
g(E[Y]) = β₀ + f₁(x₁) + f₂(x₂) + ... + fₚ(xₚ)
```
其中 g(·) 为链接函数（Logistic GAM 使用 logit 链接，Linear GAM 使用恒等链接），fⱼ(·) 为第 j 个变量的平滑函数。

**1.2 平滑性假设**
每个 fⱼ(·) 是通过B样条基函数展开的平滑函数，假设风险因素与疾病状态之间呈平滑的非线性关系，而非突变或阶梯变化。

**1.3 独立性假设**
各观测样本之间相互独立。体检数据为横截面数据，来自不同个体。

### 2. Logistic GAM 专用假设

**2.1 二项分布假设**
目标变量 DM（是否糖尿病）服从伯努利分布/二项分布：
```
Yᵢ ~ Binomial(nᵢ, pᵢ)
```
其中 pᵢ = P(DM=1 | Xᵢ) 为第 i 个个体的患病概率。

**2.2 Logit 链接函数**
```
log(p / (1-p)) = η = β₀ + Σ fⱼ(xⱼ)
```
假设对数胜算比与预测变量之间呈可加关系。

### 3. 数据质量假设

**3.1 缺失机制假设**
数据缺失假定为随机缺失（Missing at Random, MAR），即缺失与否仅依赖于已观测到的变量，可使用中位数填补。

**3.2 同分布假设**
附件1（训练集）与附件2（预测集）来自同一总体分布，即两个数据集的协变量分布一致。

**3.3 测量无偏假设**
各项体检指标的测量误差相互独立且期望为零，不会系统性偏向某一方向。

### 4. 交叉验证假设

**4.1 数据可交换性假设**
数据在交叉验证的随机划分下是可交换的，即每次划分的训练集和验证集来自同一分布。

**4.2 无信息泄露假设**
在嵌套交叉验证中，每个样本的预测值来自一个完全未见过该样本的模型，确保评估无偏。

### 5. LASSO 回归假设

**5.1 线性假设**
各预测变量与血糖值之间存在近似线性关系（适用于单因素筛选阶段）。

**5.2 稀疏性假设**
在众多候选变量中，真正对血糖有显著影响的变量个数是有限的（LASSO利用L1正则化实现变量选择）。

### 6. 统计检验假设

**6.1 Mann-Whitney U检验假设**
两独立样本来自分布形状相似的总体（位置参数可能存在差异），不要求正态性。

**6.2 卡方检验假设**
分类变量的观测频数与期望频数的差异服从卡方分布，要求各单元格期望频数不小于5。

### 7. 临床假设

**7.1 糖尿病诊断标准**
采用 WHO 标准，以空腹血糖 > 7.0 mmol/L 作为糖尿病诊断阈值。

**7.2 风险分层合理性假设**
基于预测概率的五档分位数分层能够有效区分不同风险水平的个体，每一档内的个体风险同质、档间异质。

**7.3 高危亚群定义假设**
年龄>55岁且TG>2.3 mmol/L 的复合指标可以作为高危亚群的识别标准，该阈值基于临床指南和本数据分布特征确定。
