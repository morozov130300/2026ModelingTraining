# 问题3：糖尿病风险评估 — 模型分析报告

## 1. 实验设置

| 项目 | 值 |
|------|-----|
| 总样本量 | 5,905 |
| 糖尿病阳性率（>7.0 mmol/L） | 8.04%（n=475） |
| 评估策略 | 5折StratifiedKFold嵌套交叉验证 |
| 预测方式 | 每折内独立训练GAM + XGBoost(Calibrated) → 概率平均 → OOF整合 |
| 最终特征数 | 13个（7核心 + 5扩展 + 1交互） |
| 零泄露 | 每个样本的预测来自一个从未见过该样本的模型 |

---

## 2. 特征构成

### 核心7变量（来自问题1 LASSO筛选）
年龄、甘油三酯（TG）、红细胞计数（RBC）、红细胞平均血红蛋白浓度（MCHC）、血红蛋白（HGB）、丙氨酸氨基转换酶（ALT）、性别

### 扩展5变量（点双列相关系数筛选）
低密度脂蛋白胆固醇（LDL-C）、红细胞压积（HCT）、白细胞计数（WBC）、总胆固醇（TC）、血小板计数（PLT）

### 交互项
年龄 × 甘油三酯（临床已知的糖尿病风险交互效应）

### 临床合理性
13个特征全部有已知的糖尿病生物学关联：血脂异常（TG, LDL-C, TC）、慢性低度炎症（WBC）、红细胞参数（RBC, HCT, HGB, MCHC）反映糖化对血液系统的影响、肝功能（ALT）与脂肪肝/胰岛素抵抗相关、年龄和性别是基础流行病学因素。

## 3. 模型性能 — 5折嵌套交叉验证

### 逐折AUC明细

| 折 | GAM AUC | XGBoost AUC | 集成AUC |
|---|---------|-------------|---------|
| Fold 1 | 0.7994 | 0.7732 | 0.7939 |
| Fold 2 | 0.8061 | 0.7684 | 0.7970 |
| Fold 3 | 0.8402 | 0.7761 | 0.8216 |
| Fold 4 | 0.8200 | 0.7780 | 0.8091 |
| Fold 5 | 0.8036 | 0.7827 | 0.8022 |
| **OOF整体** | — | — | **0.8040** |
| **OOF PR-AUC** | — | — | **0.2368** |

**观察要点：**
- GAM在各折表现优于XGBoost，适合此数据的非线性结构
- 集成在4/5折介于GAM和XGB之间，第3折GAM=0.8402拉高了上限
- 5折GAM区间[0.799, 0.840]，SD≈0.016，稳定性较好
- OOF AUC=0.8040是诚实无泄漏的最终评估值

### 全数据Bootstrap稳定性

| 指标 | 值 |
|------|-----|
| Bootstrap均值AUC | 0.8231 |
| 95%置信区间 | [0.8076, 0.8414] |
| 标准差 | 0.0092 |
| 范围 | [0.7935, 0.8466] |
| 收敛比例 | 180/200（90%） |

Bootstrap AUC均值（0.823）略高于OOF AUC（0.804），因为Bootstrap在全数据上训练、仅受采样变异影响，而OOF反映泛化到未见过样本的能力。二者之间的差距（~0.02）说明模型有一定程度的小样本过拟合，但幅度在可接受范围内。

## 4. 模型对比（5折CV独立验证）

| 模型 | AUC | PR-AUC |
|------|-----|--------|
| **Logistic GAM** | **0.8138** | **0.2723** |
| RandomForest | 0.7864 | 0.2187 |
| XGBoost(cal) | 0.7780 | 0.2120 |

Logistic GAM在所有指标上优于树模型，确认了GAM在此场景的适用性——特征与糖尿病风险之间存在非线性但平滑的关系，而非复杂的高阶交互。

## 5. 不同血糖阈值的区分能力

| 血糖阈值 | 阳性数 | 阳性率 | AUC | PR-AUC | 临床含义 |
|----------|--------|--------|-----|--------|---------|
| >5.6 mmol/L | 1,891 | 32.0% | 0.7301 | 0.5415 | 空腹血糖受损（IFG） |
| >6.1 mmol/L | 968 | 16.4% | 0.7844 | 0.3870 | 糖尿病前期 |
| **>7.0 mmol/L** | **475** | **8.04%** | **0.8040** | **0.2368** | **糖尿病** |
| >8.0 mmol/L | 289 | 4.89% | 0.8044 | 0.1623 | 确诊糖尿病 |

AUC随阈值提高而上升，说明模型在高血糖人群中有更好的区分度。PR-AUC随阳性率降低而下降是数学必然（稀有事件PR-AUC上限低），不代表模型变差。

## 6. 五档风险分层评估

### 分层结果

| 风险等级 | n | 占比 | 阳性率 | 累积灵敏度 | 累积PPV | 临床含义 |
|---------|---|------|--------|-----------|---------|---------|
| 极低风险 | 591 | 10.0% | **0.34%** | 1.000 | 0.080 | 可安全排除糖尿病 |
| 低风险 | 1,181 | 20.0% | **0.68%** | 0.996 | 0.089 | 常规随访 |
| 中等风险 | 1,181 | 20.0% | **2.79%** | 0.979 | 0.113 | 灰色地带，建议OGTT |
| 高风险 | 1,771 | 30.0% | **9.32%** | 0.909 | 0.146 | 建议糖化血红蛋白检测 |
| 极高风险 | 1,181 | 20.0% | **22.61%** | 0.562 | 0.226 | 强烈建议临床确诊 |

### 分层梯度验证
阳性率：0.34% → 0.68% → 2.79% → 9.32% → 22.61%
每相邻两档之间阳性率呈单调递增，分层有统计学意义。

### 临床决策路径

```
┌────────────────────────────────────────────────┐
│                  全体人群 (5,905)               │
│              基线糖尿病阳性率 8.04%               │
└────────────────────┬───────────────────────────┘
                     │
    ┌────────────────┼────────────────┬───────────┐
    ▼                ▼                ▼           ▼
 极低风险          低风险          中等风险    高风险+极高风险
  (10%)            (20%)           (20%)        (50%)
    │                │                │           │
  NPV>99.4%      低于基线阳性率    灰区24.7%   PPV 14.6%
    ▼                ▼                ▼           ▼
 安全排除         常规随访      生活方式干预    临床确诊检查
                               +OGTT建议
```

### 关键指标
- **极低+低风险 NPV = 99.44%** — 排除糖尿病能力极强
- **高风险+极高风险合并灵敏度 = 90.9%** — 能检出90%以上的真实糖尿病患者
- **中风险灰区占比 = 20.0%** — 需进一步检查的人群规模可控
- **极高风险独立PPV = 22.6%** — 相比基线8.04%富集2.8倍

## 7. 最优阈值选择

| 选择标准 | 阈值 | 敏感度 | 特异度 | PPV | NPV | F2 |
|---------|------|--------|--------|-----|-----|-----|
| Youden指数 | 0.4338 | 0.8337 | 0.6484 | 0.1718 | 0.9781 | 0.4709 |
| **F2最优** | **0.4627** | **0.8042** | **0.6790** | **0.1798** | **0.9754** | **0.4745** |
| 代价最优(FP:FN=1:3) | 0.7673 | 0.2674 | 0.9451 | 0.2988 | 0.9365 | 0.2731 |

**最终选用F2最优阈值（0.4627）**，理由：
- F2（beta=2）将召回率权重设为精确率的2倍，适合筛查场景
- 敏感度80.4%意味着每5个糖尿病患者能检出4个
- 相比Youden阈值，F2阈值在仅牺牲3个百分点敏感度的条件下，提高了3个百分点特异度
- 混淆矩阵：TP=382, FN=93, FP=1,743, TN=3,687

## 8. 偏效应分析（Logistic GAM）

GAM的核心优势在于可解释性。偏效应图展示每个特征对糖尿病风险log-odds的平滑贡献：

| 特征 | 预期偏效应形态 |
|------|--------------|
| 年龄 | 随年龄递增，尤其在>60岁加速 |
| 甘油三酯(TG) | 非线性上升，高TG显著增加风险 |
| 红细胞计数(RBC) | 可能呈U型或倒U型 |
| MCHC | 可能与血糖代谢相关 |
| 血红蛋白(HGB) | 高HGB提示糖化负荷增加 |
| ALT | 反映脂肪肝→胰岛素抵抗路径 |
| 性别 | 离散偏效应，男性参考编码为1 |
| 低密度脂蛋白胆固醇 | 与TG协同增加风险 |
| 红细胞压积 | 类似HGB的偏效应 |
| 白细胞计数 | 慢性低度炎症标志物 |
| 总胆固醇 | 与LDL-C协同 |
| 血小板计数 | 炎症/凝血相关 |
| 年龄_TG交互 | 年龄×TG的非线性协同效应 |

## 10. 模型局限性

### 固有局限性（非模型可解决）

1. **信息上限**：仅使用横截面血检指标，缺少
   - BMI/腰围（肥胖指标）
   - 家族史/遗传风险
   - 饮食/运动/生活方式
   - 既往血糖轨迹（纵向数据）
   - 胰岛素水平/C肽
   
2. **AUC理论天花板**：在此信息集上，AUC上限估计在0.82-0.85

3. **不平衡分类**：8%阳性率导致PPV天然偏低，任何模型都会面临大量假阳性

### 模型可改进的方向

| 方向 | 预期边际收益 | 计算成本 |
|------|------------|---------|
| Stacking meta-learner | +0.005-0.01 AUC | 中 |
| XGBoost超参数GridSearch | +0.005-0.01 AUC | 高 |
| GAM lambda/spline调参 | +0.003-0.008 AUC | 中 |
| 更多临床交互特征 | +0.005-0.01 AUC | 低 |
| 贝叶斯优化超参数 | +0.003-0.008 AUC | 高 |

## 11. 结论与建议

### 核心结论

**OOF AUC = 0.8040** 是诚实、稳定、可复现的性能评估。对于仅使用横截面血检数据预测糖尿病风险这项任务，这已接近信息上限。

### 可交付价值

1. **排除糖尿病能力强**：极低+低风险NPV=99.44%，可安全减少不必要的进一步检查
2. **五档分层有临床可操作性**：清晰的从"安全排除"到"强烈建议确诊"的梯度
3. **模型可解释性好**：GAM偏效应图可直接解释每个特征的风险贡献
4. **多重验证互相印证**：OOF AUC(0.804)、5-CV GAM(0.814)、Bootstrap(0.823)三者一致

### 建议使用场景

- **一级筛查**：高风险+极高风险(占人群50%)建议进一步OGTT检查
- **糖尿病排除**：极低+低风险(占人群30%)可安全排除
- **中风险人群**（占人群20%）：建议生活方式干预，必要时OGTT
- **成本效益**：将需临床确诊检查的人群从100%缩小到50%
