## 第一部分：符号说明

### 1.1 通用符号

| 符号 | 含义 | 出处 |
|------|------|------|
| \(n\) | 样本量（训练集 5905，附件2 141） | 所有文件 |
| \(p\) | 特征数量 | 所有文件 |
| \(X\) | 特征矩阵，\(n \times p\) | 所有文件 |
| \(y\) | 目标变量/真实标签 | 所有文件 |
| \(\hat{y}\) | 模型预测值 | t2.py, t3.py |
| AUC | ROC曲线下面积（区分度指标） | t3.py, t4.py |
| PR-AUC | 精确率-召回率曲线下面积 | t3.py, 报告 |
| RMSE | 均方根误差 \(\sqrt{\frac{1}{n}\sum(y_i-\hat{y}_i)^2}\) | t2.py |
| \(R^2\) | 决定系数 \(1 - SS_{res}/SS_{tot}\) | t2.py, 数据预处理.py |
| \(\bar{R}^2\) | 调整\(R^2\) \(= 1-(1-R^2)\frac{n-1}{n-p-1}\) | t2.py |
| AIC | 赤池信息准则 | t2.py, 数据预处理.py |
| BIC | 贝叶斯信息准则 | 数据预处理.py |
| GCV | 广义交叉验证得分 | t2.py |

| EDF / edof | 有效自由度（GAM平滑项总自由度） | t2.py, t3.py |
| SE | 标准误 | 数据预处理.py |
| SD | 标准差 | 所有文件 |
| CI | 置信区间（95%: 2.5–97.5百分位数） | t3.py, 报告 |
| \(\lambda\) | LASSO正则化强度 / GAM平滑惩罚参数 | t1.py, t2.py, t3.py |

### 1.2 分类模型评估符号

| 符号 | 含义 | 公式 |
|------|------|------|
| TP | 真阳性（实际患病且预测患病） | — |
| TN | 真阴性（实际未患病且预测未患病） | — |
| FP | 假阳性（实际未患病但预测患病） | — |
| FN | 假阴性（实际患病但预测未患病） | — |
| Sens / TPR | 敏感度（召回率） | \(TP/(TP+FN)\) |
| Spec | 特异度 | \(TN/(TN+FP)\) |
| PPV | 阳性预测值（精确率） | \(TP/(TP+FP)\) |
| NPV | 阴性预测值 | \(TN/(TN+FN)\) |
| F1 | F1分数 | \(2TP/(2TP+FP+FN)\) |
| \(F_\beta\) | F-beta分数 | \((1+\beta^2)\frac{P\cdot R}{\beta^2 P+R}\) |
| Youden \(J\) | Youden指数 | \(Sens + Spec - 1\) |
| NB | 净收益（决策曲线分析） | \(TP/n - FP/n \cdot p_t/(1-p_t)\) |
| OR | 胜算比 | \(p/(1-p)\) |
| log(OR) | 对数胜算比 | \(\ln(p/(1-p))\) |
| \(\Delta\)logOR | 贡献度（变量在整个取值范围内log(OR)的极差） | \(\max\eta(x)-\min\eta(x)\) |

### 2.1 数据预处理假设

| 编号 | 假设 | 说明 | 出处 |
|------|------|------|------|

| A3 | **众数代表分类变量典型值** | 分类变量缺失用众数填补 | 数据预处理.py |

| 编号 | 假设 | 说明 | 出处 |

### 2.3 问题2：血糖预测GAM假设

| 编号 | 假设 | 说明 | 出处 |
|------|------|------|------|
| C1 | **可加性** | 模型A：\(\text{血糖} = \beta_0 + \sum f_j(X_j) + \varepsilon\) | t2.py |
| C2 | **可加性（交互模型）** | 模型B：\(\text{血糖} = \beta_0 + f_1(\text{年龄}) + f_2(\text{TG}) + f_{12}(\text{年龄},\text{TG}) + \sum_{j\geq3} f_j(X_j) + \varepsilon\) | t2.py |
| C3 | **平滑性** | 每个\(f_j\)是光滑函数，用P-样条逼近（\(k=10\)） | t2.py |

| C5 | **残差正态性** | \(\varepsilon \sim N(0,\sigma^2)\)，通过Q-Q图和Shapiro-Wilk检验验证 | t2.py |
| C6 | **残差同方差性** | \(Var(\varepsilon|X)=\sigma^2\)常数，通过残差-拟合值图检验 | t2.py |
| C7 | **残差独立性** | 各观测残差不相关，通过ACF自相关图检验 | t2.py |
| C8 | **观测独立** | 各体检记录相互独立 | 所有模型 |
| C9 | **ΔAIC>2为显著改进** | 交互模型AIC比加性模型低2以上则认为交互项显著 | t2.py |
| C10 | **GCV选择平滑参数** | Generalized Cross-Validation自动选择最优平滑度 | t2.py |
| C11 | **10折交叉验证评估泛化能力** | 每折独立训练和测试 | t2.py |

| 编号 | 假设 | 说明 | 出处 |
