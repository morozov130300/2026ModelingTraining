# 武汉理工大学数学建模训练 — 糖尿病风险预测（C题）

## 问题1：变量筛选与分析

基于 LASSO 回归从 44 个体检指标中筛选出与血糖（GLU）最相关的 **5 个主要变量**，建立多元线性回归预测模型。

### 核心方法

| 步骤 | 方法 | 说明 |
|------|------|------|
| 缺失值处理 | **MICE多重插补**（IterativeImputer） | 分层策略：MICE+中位数+缺失指示变量 |
| 单因素筛选 | Pearson/Spearman相关（p<0.05） | 初步筛选与血糖相关的指标 |
| 变量压缩 | **LASSO回归**（LassoCV交叉验证） | L1正则化自动筛选5个主要变量 |
| 回归验证 | 多元线性回归（p值+95%CI） | 验证变量显著性并建模 |

### 最终筛选变量

| 变量 | 全称 | 生理系统 | 系数方向 |
|------|------|---------|---------|
| age | 年龄 | 基本信息 | + |
| TG | 甘油三酯 | 血脂代谢 | + |
| BUN | 尿素 | 肾功能 | + |
| HGB | 血红蛋白 | 血常规 | + |
| ALT | 丙氨酸氨基转换酶 | 肝功能 | + |

模型表现：**R²=0.130，F=177.0，5变量全部p<0.001**
